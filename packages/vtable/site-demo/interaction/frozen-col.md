# 基本表格

选中单元格 配置选中效果

## 关键配置

 - `allowFrozenColCount`  可选做冻结的前几列 默认0
 - `frozenColCount`  初始冻结列数 默认0
 - `showFrozenIcon` 是否显示冻结图标 默认true
