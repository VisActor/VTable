# 透视表多列树形展示

透视表格多列树形展示。多列属于特殊的树形展示结构，需要依赖业务方传入 rowTree，columnTree 和 extensionRows。

## 关键配置

- `rowHierarchyType` 将层级展示设置为`tree`，默认为平铺模式`grid`。
- `extensionRows` 设置扩展行表头，当且进度`rowHierarchyType`为`tree`时生效。
