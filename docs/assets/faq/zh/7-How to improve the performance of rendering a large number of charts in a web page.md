# 怎么提高页面中大量图表渲染的性能？特别是在滚动交互过程中。

## 问题描述

我的使用场景是需要批量绘制大量图表，不过这些图表具有一定的规律性，整行或者整列是同类型图表，有什么方式可以优化性能呢？现在问题是当滚动的时候，图表渲染跟不上，滚动交互也不流畅。

## 解决方案

在VTable中，使用 VChart 来绘制单元格内图表，内部做了渲染优化，可以解决这个问题。使用方式是如下：

## 代码示例
