Simple List Table

Create a basic data table with columns and rows. This is the most common use case for displaying tabular data.

ID Name Age City Salary
1 John Doe 30 New York $75,000
2 Jane Smith 25 Los Angeles $80,000
3 Bob Johnson 35 Chicago $70,000
4 Alice Brown 28 Houston $85,000 
// VTable Basic Configuration
const option = {
    container: document.getElementById('table'),
    records: [
        { id: 1, name: 'John Doe', age: 30, city: 'New York', salary: 75000 },
        { id: 2, name: 'Jane Smith', age: 25, city: 'Los Angeles', salary: 80000 },
        // ... more data
    ],
    columns: [
        { field: 'id', title: 'ID', width: 80 },
        { field: 'name', title: 'Name', width: 150 },
        { field: 'age', title: 'Age', width: 80 },
        { field: 'city', title: 'City', width: 120 },
        { field: 'salary', title: 'Salary', width: 100 }
    ]
};

const table = new VTable.ListTable(option);

Conditional Styling

Apply different styles based on data values to highlight important information.

Employee Department Salary Status
Alice Johnson Engineering $85,000 Active
Bob Smith Marketing $72,000 Active
Carol Davis Engineering $92,000 Inactive
David Wilson Sales $68,000 Active 
// Conditional Styling Example
{
    field: 'salary',
    title: 'Salary',
    width: 100,
    style: {
        bgColor(args) {
            const { dataValue } = args;
            if (dataValue >= 80000) return '#d4edda'; // Green for high salary
            if (dataValue >= 75000) return '#fff3cd'; // Yellow for medium
            return '#f8d7da'; // Red for low salary
        },
        color(args) {
            const { dataValue } = args;
            if (dataValue >= 80000) return '#155724';
            if (dataValue >= 75000) return '#856404';
            return '#721c24';
        }
    }
}

Learn how to fully customize every aspect of your VTable to meet specific requirements.

Getting Started

// Installation
npm install @visactor/vtable

// Basic Import
import * as VTable from '@visactor/vtable';

// Create a simple table
const option = {
    container: document.getElementById('tableContainer'),
    records: yourData,
    columns: [
        { field: 'name', title: 'Name', width: 150 },
        { field: 'age', title: 'Age', width: 80 },
        { field: 'city', title: 'City', width: 120 }
    ]
};

const table = new VTable.ListTable(option);

Key Configuration Options:

  • Theme Customization: Complete control over colors, fonts, and styling
  • Cell Types: Text, numbers, progress bars, charts, images, and custom renderers
  • Interactive Features: Sorting, filtering, editing, selection, and drag & drop
  • Performance: Virtual scrolling, lazy loading, and efficient rendering for large datasets
  • Events: Comprehensive event system for user interactions and data changes

Common Use Cases:

  • Data dashboards and business intelligence
  • Admin panels and content management
  • Financial reports and analytics
  • E-commerce product catalogs
  • Scientific data visualization
